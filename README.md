# dle-bbcode
Плагин возвращающий убранный BBcode editor. Актуально для версий **18.0**, **18.1**, **19.0**, **19.1**.

Других версий и поддержки в других версиях как минимум от меня не будет. В 20й версии DLE уже начинается удаление всяких старых остаточных механизмов и изменение существующих, адаптировать данный редактор как плагин больше не имеется возможности и желания.
